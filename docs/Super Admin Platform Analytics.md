# Super Admin Platform Analytics

## Feature Overview
A high-level network intelligence dashboard designed for super administrators to monitor the health and growth of the entire platform.

## Implementation Details
- **Page Path**: `/super-admin/analytics`
- **Component**: `SuperAdminAnalyticsPage`
- **Key Sections**:
  - **Platform Growth Metrics**: Real-time stats for registered masjids, active users, and transaction volume.
  - **Network Intelligence Charts**: Large-scale visualizations of global transaction volume.
  - **System Hub Activity**: Live feed of network events, including new registrations and infrastructure alerts.

## UI Design
- Professional dark theme (`zinc-950`) with emerald-400 highlights.
- Modern "Network Intelligence" aesthetic with blurred background gradients.
- Custom sidebar for infrastructure and identity management.

## Technical Notes
- High-performance layout for real-time monitoring.
- Modular metrics grid.
