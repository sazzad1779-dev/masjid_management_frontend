# Reports & Analytics Dashboard

## Feature Overview
Comprehensive system analytics providing a high-level overview of institutional financial health and stewardship reporting.

## Implementation Details
- **Page Path**: `/analytics`
- **Component**: `AnalyticsPage`
- **Key Sections**:
  - **Audited Financial Summary**: Interactive bar chart showing monthly income/expense trends for the fiscal year.
  - **Category Breakdown**: Linear progress indicators for different income and expense categories (e.g., General Donation, Zakat, Utilities, Payroll).
  - **Institutional Quick Stats**: Highlights for donor growth and year-over-year financial comparisons.
  - **Report Generation**: Shortcuts for generating annual and quarterly audited statements.

## UI Design
- Dark-themed hero section for high-impact reporting items.
- Modern rounded-3xl cards for analytics modules.
- Consistent emerald-green accent system.

## Technical Notes
- CSS-based bar chart implementation for consistent rendering.
- Scalable category breakdown components.
