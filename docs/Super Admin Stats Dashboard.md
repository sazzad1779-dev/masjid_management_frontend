# Super Admin Stats Dashboard

## Feature Overview
The Super Admin Stats Dashboard provides real-time statistics and multi-tenant management capabilities for the entire institutional network.

## Implementation Details
- **Page Path**: `/super-admin/stats`
- **Component**: `SuperAdminStatsPage`
- **Key Sections**:
  - **Global Overview Stats**: Large-scale metrics for total masjids, users, and platform-wide revenue.
  - **Institutional Directory**: Comprehensive table for managing registered institutions, their locations, and financial performance.
  - **Infrastructure Health Monitor**: Real-time status indicators for system nodes.

## UI Design
- Ultra-modern dark aesthetic with premium typography and gradients.
- High-contrast revenue tracking.
- Interactive directory with tenant-level status indicators.

## Technical Notes
- Optimized for multi-tenant management views.
- Secure infrastructure monitoring interface.
